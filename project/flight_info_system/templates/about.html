About Me

This website is designed by Rongrong(Spencer) Liu. He is currently a master of computer science student at University of Virginia. Previous to that, he studied and lived in Shanghai.

The website is developed using Python, with Django as backend and Bootstrap as frontend. A hand-drawn look style sheet named Sketchy is used to replace the original Bootstrap widgets.

A lot of functions is still under development.
